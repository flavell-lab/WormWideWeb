Data Processing & Analysis Code

All software developed for this project is available below, including:

  • • The ANTSUN pipeline for extracting neural traces from GCaMP volumes
  • • The CePNEM model for neural trace analysis and behavioral encoding
  • • Stage control software for tracking worms during imaging
  • • Example Jupyter notebooks for data analysis and plotting
  • • Additional downloadable data for neural network weights, posterior distributions, and more

Atanas & Kim et al., 2023 Datasets

If you'd like to access full datasets from the original study releasing this website, please refer to the following data repository.

You can download all datasets at once, and the HDF5 files contain all information (i.e. behavior variables that were not included in this website) for each animal.

processed_h5.tar.bz2 contains all datasets from Atanas & Kim et al., 2023. in HDF5.
